## 技术简介：
*   核心框架：.NET 5.0, ASP.NET Core Runtime 5.0
*   持久层框架：EntityFramework Core 5.0
*   日志框架：Serilog 2.10, LogDashboard 1.4
*   性能监控：MiniProfiler 4.2
